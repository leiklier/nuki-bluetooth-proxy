# Nuki Opener BLE for Home Assistant

Control a [Nuki Opener](https://nuki.io/en/opener/) from Home Assistant **directly over
Bluetooth** — no Nuki Bridge and no dedicated ESP32 required. If you already have
[ESPHome Bluetooth proxies](https://esphome.io/components/bluetooth_proxy.html) (or a local
Bluetooth adapter), this integration talks the native Nuki BLE protocol through them.

Inspired by [nuki_hub](https://github.com/technyon/nuki_hub); protocol implemented from the
official Nuki Opener BLE API specification.
